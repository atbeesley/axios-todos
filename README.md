# axios-todos

## In the root of the project, run:
```
npm install
```

### To start the server::
```
npm run serve
```

### In a new terminal, run:
```
npm run backend
```

### Refresh the browser to see the seeded todos, or add your own!



### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
